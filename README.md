# Use your noodles
## Abstract
This repo contains the code used to solve the challenge *Use your noodles*. This challenge was given as part of the interviews for Engineered Arts.
 
![image](https://github.com/adiego73/noodle-challenge/blob/master/assets/noodles.png?raw=true)

The idea is to create an image of a bowl full of noodles. The noodles should be of different lengths, with random curvatures, and should not exceed the perimeter of the bowl.

The result can be seen in the following image:

![result](https://github.com/adiego73/noodle-challenge/blob/master/assets/result.png?raw=true)

A running example can be found [here](https://adiego73.github.io/noodle-challenge/).
